## Reading data 
* Reading and writing tabular data - dataframes and tables
    #### Reading
    ```
    read.table
    ```
    ```
    read.csv
    ```
    
    #### Writing
    ```
    write.table
    ```
    ```
    write.csv
    ```

* Reading and writing lines of a text file (abc.txt)
    #### Reading
    ```
    readLines
    ```
    
    #### Writing
    ```
    writeLines
    ```
